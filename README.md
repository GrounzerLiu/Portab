# Portab

新标签页替换扩展 —— 搜索框 + 按访问次数排列的历史记录网格。

## 功能

- **搜索** — 搜索框，搜索建议
- **常用网站** — 按访问次数自动排列，支持固定到顶部区域
- **主题** — 暗色/亮色/跟随系统，自定义主题色（8 种预设 + 取色器），自动从壁纸取色
- **壁纸** — 无/必应每日/随机（Lorem Picsum）/自定义图片，支持遮罩和模糊
- **时钟** — 右键自定义字号、字体、边距
- **布局** — 可调节列数、行数、间距
- **历史范围** — 7天/30天/90天/全部
- **站点屏蔽** — 不在主页显示的站点管理
- **Favicon 缓存** — 三级回退链，避免重复请求

## 安装

1. 打开 `chrome://extensions`
2. 开启**开发者模式**
3. 点击**加载未打包的扩展程序**
4. 选择 `Portab` 文件夹

## 技术

- Manifest V3
- Material Design 3 视觉风格（Material Color Utilities 取色）
- 所有图标使用 Material Symbols 内联 SVG
